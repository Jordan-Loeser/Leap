Columns

A Responsive Grid System

An Example

Leap uses a column system that is determined using CSS classes. Using classes, you can decide how wide you want your column to be on each device.

On Mobile Devices

The example below uses a 12-column layout. On an extra small (xs) device, such as a cellphone, both the left and right bars utilize 6/12 columns.

On Larger Devices

Once the device reaches a small (sm) size, such as a tablet, the columns adjust. The left bar now uses 8/12 columns, and the right bar uses 4/12 columns. See more examples here.
